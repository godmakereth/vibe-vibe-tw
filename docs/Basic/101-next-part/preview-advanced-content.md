# 進階版核心內容詳解

想知道進階版具體會教什麼？這裏是詳細介紹。


## 第 0 章：Bootcamp —— 零基礎概念打底

**爲什麼需要這一章？**

進階版會涉及更多"技術性"的內容。這一章幫你建立必要的認知基礎。

**你將學到：**

| 小節 | 內容 |
|:-----|:-----|
| 0.0 | 課程概念定義：什麼是全棧開發 |
| 0.1 | 計算機基礎：文件系統、進程、端口 |
| 0.2 | 命令行入門：像黑客一樣操作電腦 |
| 0.3 | 前端三件套：HTML、CSS、JavaScript |
| 0.3.5 | 網絡基礎：HTTP、域名、DNS |
| 0.4 | 開發環境配置：Node.js、包管理器 |
| 0.5 | TypeScript 入門：從動態到靜態類型 |
| 0.6 | 開發安全底線：環境變量與密鑰管理 |
| 0.7 | Docker 核心概念：容器化入門 |

::: info 💡 如果你有編程基礎
可以根據自身情況跳過部分章節。但建議至少瀏覽一遍 0.5（TypeScript）和 0.7（Docker）。
:::


## 第 1 章：概念與準備 —— 從思維到工具

**這一章做什麼？**

把 Bootcamp 的認知轉化爲可執行的行動。搭建完整的開發環境，跑通第一個 Next.js 項目。

**你將學到：**

- 工具裝配與驗證：選擇 IDE、配置 AI 助手
- Vibe Coding 心法進階：如何"指揮" AI 而不是被 AI 牽着走
- 最小上線實戰：用 1Panel 完成第一次部署
- AI 協作工作流：建立可複用的人機協作模式

**核心目標：**

✅ 擁有一套完整的開發環境

✅ 跑通第一個 Hello World

✅ 完成第一次部署，擁有一個全世界都能訪問的 URL


## 第 2 章：技術選型與架構藍圖

**這一章做什麼？**

理解爲什麼選擇 Next.js + TypeScript + Prisma 這套技術棧。瞭解現代全棧應用的架構設計。

**你將學到：**

- 技術棧全景與版本管理策略
- Next.js App Router 架構詳解
- 渲染策略：CSR / SSR / SSG / ISR 的區別與選擇
- 前後端協作與 API 契約
- 架構分層：表現層 / 接口層 / 業務層 / 數據層
- Supabase 適用場景與取捨

**核心理解：**

> 技術選型的本質不是選"最好的"，而是選"最適合的"。


## 第 3 章：前端開發 —— 把頁面跑起來

**這一章做什麼？**

從基礎的 React 組件到完整的頁面，掌握現代前端開發的核心技能。

**你將學到：**

- React 組件基礎：函數組件、JSX、Props
- 狀態管理：useState、useEffect
- 路由系統：Next.js 的文件系統路由
- UI 組件庫：shadcn/ui 實戰
- 樣式方案：Tailwind CSS 原子化 CSS


## 第 4 章：數據庫與數據 —— 數據生命週期管理

**這一章做什麼？**

從簡單的數據存儲到完整的數據庫設計，掌握數據管理的核心能力。

**你將學到：**

- PostgreSQL 基礎：關係型數據庫入門
- Prisma ORM：類型安全的數據庫操作
- 數據建模：如何設計數據表結構
- CRUD 操作：增刪改查的最佳實踐
- 數據驗證：保證數據質量


## 第 5 章：產品與文檔 —— 讓人和 AI 都讀得懂

**這一章做什麼？**

從基礎版的簡單 PRD 到更專業的文檔體系，讓你的項目可維護、可協作。

**你將學到：**

- PRD 進階：更完整的需求文檔
- 技術文檔：架構文檔、API 文檔
- 文檔自動生成：讓 AI 幫你寫文檔
- 版本管理：文檔的版本控制


## 第 6 章：認證與安全 —— 權限管理

**這一章做什麼？**

實現用戶註冊、登錄、權限控制，保護你的應用和用戶數據。

**你將學到：**

- 認證 vs 授權：兩個概念的區別
- JWT 和 Session：兩種認證方式
- NextAuth.js：開箱即用的認證方案
- 權限控制：基於角色的訪問控制（RBAC）
- 安全最佳實踐：防範常見攻擊


## 第 7 章：API 設計規範

**這一章做什麼？**

設計規範、易用、安全的 API 接口。

**你將學到：**

- RESTful 設計原則
- 請求與響應格式規範
- 數據校驗：Zod 驗證庫
- 錯誤處理：統一的錯誤響應
- API 版本管理


## 第 8 章：項目規則與協作

**這一章做什麼？**

建立規範的項目管理流程，爲團隊協作做好準備。

**你將學到：**

- Git 工作流：分支策略、提交規範
- 代碼規範：ESLint、Prettier
- 代碼審查：Pull Request 流程
- 項目結構：目錄組織最佳實踐


## 第 9 章：測試與質量

**這一章做什麼？**

用自動化測試保證代碼質量，減少 Bug。

**你將學到：**

- 爲什麼需要測試
- 單元測試：測試單個函數/組件
- 集成測試：測試多個模塊協作
- E2E 測試：測試完整用戶流程
- 測試覆蓋率與持續集成


## 第 10 章：部署與運維

**這一章做什麼？**

把你的應用部署到生產環境，並保持穩定運行。

**你將學到：**

- Vercel 部署：零配置上線
- Docker 容器化：打包你的應用
- CI/CD 流水線：自動化部署
- 環境管理：開發/測試/生產環境
- 監控與告警：發現問題、快速響應


## 第 11 章：發佈與覆盤

**這一章做什麼？**

正式發佈你的產品，收集反饋，持續迭代。

**你將學到：**

- 版本發佈流程
- 用戶反饋收集
- 數據分析入門
- 迭代優化方法論
- 覆盤總結：從項目中學習


## 第 12 章：高級專題 —— 進階優化

**這一章做什麼？**

深入探索高級主題，讓你的應用從"能用"變成"好用"。

**你將學到：**

- SEO 優化：讓搜索引擎找到你
- 性能優化：讓應用更快
- AI SDK 集成：在應用中使用 AI 能力
- 即時通信：WebSocket 入門
- 大文件處理：文件上傳與下載
- 國際化：支持多語言


## 小結

進階版不是要把你變成"專業程序員"。

而是讓你在 Vibe Coding 的基礎上，掌握更多工具，解決更復雜的問題。

**你依然是"指揮官"，只是指揮的範圍更大了。**
