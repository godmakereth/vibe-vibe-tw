# 4.1.2 準備你的 AI IDE

> **本小節目標**：用 5 分鐘確認你的 AI IDE 已準備就緒

## 確認工具已安裝

如果你跟着第一章學習，應該已經選擇並安裝了一個 AI IDE。

還沒安裝？請先回顧 [1.4 工具選型指南](../../01-awakening/1.4-tools-guide/)，完成安裝後再繼續。

::: warning 重要
本章使用本地安裝的 AI IDE 進行開發。請確保你的電腦上已經安裝好工具。
:::

## 熟悉 AI IDE 的界面佈局

無論你使用哪款 AI IDE，它們都有相似的界面結構：

```
┌────────────────────────────────────────────────────────────┐
│                        AI IDE 通用佈局                      │
├──────────────┬─────────────────────┬───────────────────────┤
│              │                     │                       │
│   文件區      │      代碼區          │       預覽區          │
│  (項目文件)   │   (AI生成的代碼)      │   (即時效果)          │
│              │                     │                       │
├──────────────┴─────────────────────┴───────────────────────┤
│                        對話區                               │
│               (你輸入 Prompt 的地方)                         │
└────────────────────────────────────────────────────────────┘
```

### 對話區：你和 AI 溝通的地方

這是最重要的區域。你在這裏輸入 Prompt（也叫指令、需求描述），AI 會理解你的意圖並生成代碼。

**找到它**：通常在界面底部或側邊，有一個輸入框，可能標註着 "Ask AI"、"Chat"、"Composer" 等字樣。

### 代碼區：AI 生成代碼的地方

AI 生成的代碼會顯示在這裏。你通常不需要直接編輯這些代碼，但可以查看。

**找到它**：通常在界面中央，顯示着類似 `index.html`、`app.js` 這樣的文件內容。

### 文件區：管理項目文件的地方

這裏顯示項目中的所有文件和文件夾。隨着開發推進，你會看到越來越多的文件出現在這裏。

**找到它**：通常在界面左側，是一個樹狀的文件列表。

### 預覽區：看到效果的地方

這是最有成就感的區域。你的網頁會在這裏即時顯示。

**找到它**：通常在界面右側，可能需要點擊 "Preview"、"Run" 或類似按鈕纔會出現。有些 AI IDE 也支持在瀏覽器中打開預覽。

## 創建項目文件夾

在開始之前，建議在你的電腦上創建一個專門的項目文件夾：

1. 打開"文檔"或"桌面"文件夾
2. 創建一個新文件夾，命名爲 `vibe-coding-projects`
3. 在裏面再創建一個子文件夾 `todo-list`（待辦清單項目）

文件夾結構如下：

```
文檔/
└── vibe-coding-projects/
    └── todo-list/        ← 本章項目放這裏
```

::: info 💡 爲什麼要這樣做
把項目放在固定位置，方便你以後找到。AI IDE 生成的文件都會保存在這個文件夾裏。
:::

## 在 AI IDE 中打開項目文件夾

1. 打開你的 AI IDE
2. 找到"打開文件夾"或"Open Folder"選項
3. 選擇剛纔創建的 `todo-list` 文件夾
4. 確認打開

現在，你的 AI IDE 應該顯示一個空的項目。

## 驗證工具正常：一個簡單測試

在對話區輸入以下 Prompt，測試工具是否正常工作：

```
請幫我創建一個簡單的網頁，顯示"Hello, Vibe Coding!"這句話，文字居中，使用藍色。
```

### 預期結果

你應該看到：
1. AI 開始生成代碼（通常是一個 `index.html` 文件）
2. 代碼區顯示生成的 HTML 代碼
3. 文件區出現新的文件
4. 預覽區顯示一個網頁，上面有藍色居中的 "Hello, Vibe Coding!" 文字

### 如果一切正常

恭喜！你的工具已經準備好了。可以直接跳到 [4.1.4 最終檢查清單](./4.1.4-checklist.md)。

### 如果遇到問題

| 問題 | 可能原因 | 解決方法 |
|------|---------|---------|
| 對話區沒反應 | 網絡問題或未登錄 | 檢查網絡連接，確認已登錄賬號 |
| AI 回覆很慢 | 網絡延遲 | 耐心等待，或檢查網絡狀況 |
| 代碼生成了但看不到預覽 | 預覽未開啓 | 找到 "Preview" 或 "Run" 按鈕，或在瀏覽器中打開生成的 HTML 文件 |
| 提示額度用完 | 免費額度耗盡 | 查看是否可以購買額度，或等待月度額度刷新 |
| 界面全是英文 | 語言設置 | 不影響使用，AI 可以理解中文 Prompt |

## 關於免費額度

大多數 AI IDE 都提供免費額度或試用期。對於本教程的待辦清單項目，免費額度通常足夠。

如果額度不夠：
1. 查看工具是否有付費選項
2. 優化你的 Prompt，減少不必要的對話
3. 等待月度額度刷新

## 本小節要點

✓ **工具已安裝**：你的電腦上有一個可用的 AI IDE

✓ **界面已熟悉**：知道對話區、代碼區、文件區、預覽區在哪裏

✓ **項目已創建**：有一個專門的項目文件夾，AI IDE 已打開這個文件夾

✓ **測試已通過**：用 "Hello, Vibe Coding!" 驗證了工具正常工作
