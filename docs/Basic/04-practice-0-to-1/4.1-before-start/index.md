# 4.1 開始之前：整合你的準備工作

> **本節目標**：用 15 分鐘完成所有準備，然後立刻開始動手

## 經過本節學習，你將：

- 明確要做什麼（回顧第二章的思考成果）
- 確認工具就緒（你的 AI IDE 已準備好）
- 完成心態準備（準備好接受報錯、準備好不完美）
- 帶着信心進入 4.2 節，寫出第一個 Prompt

## 爲什麼需要這一節

你可能會想：我已經學了兩章，現在就想動手。

這個想法完全正確。

本節存在的目的，不是讓你"再準備準備"，而是幫你**快速確認一切就緒**。

很多人卡在"準備階段"，不是因爲真的沒準備好，而是因爲不確定自己準備好了沒有。這種不確定感會變成拖延的藉口。

本節要做的就是：**15 分鐘，把所有"我準備好了嗎"的疑問變成確定的"是"**。

## 關於"準備焦慮"

開始一個新項目時，很多人會陷入這樣的循環：

```
"我是不是應該先把工具學得更熟？"
→ "我的 PRD 是不是還不夠完整？"
→ "要不要先看幾個別人的項目？"
→ "算了，今天先準備，明天再開始..."
```

這是正常的心理反應，但它會阻止你真正開始。

**事實是**：你不需要準備到 100% 才能開始。60% 的準備加上立刻行動，比 100% 的準備加上明天再說，效果好得多。

本節會幫你確認：你已經達到了那個 60%。

## 兩種閱讀方式

### 方式一：完整閱讀（推薦新手）

按順序閱讀 4.1.1 到 4.1.4，每個小節都會幫你確認一項準備工作。

### 方式二：快速通道（適合自信的讀者）

如果你滿足以下條件，可以直接跳到 [4.1.4 最終檢查清單](./4.1.4-checklist.md)：

- 你已經完成了第二章的項目思考（或者決定跟着小李案例做）
- 你的 AI IDE 已經安裝/打開，能正常使用
- 你有 1-2 小時的專注時間

::: tip 核心觀點
準備工作的目標是"開始"，而不是"完美"。15 分鐘後，你就要寫出第一個 Prompt 了。
:::
