# 4.3.5 階段性成果檢查

> **本步目標**：確認第二輪所有功能正常，準備進入下一輪

## 第二輪完成檢查清單

在進入數據存儲（4.4）之前，請確認以下功能都能正常工作：

### 添加任務

- [ ] 輸入內容後點擊添加，任務出現在列表中
- [ ] 添加後輸入框自動清空
- [ ] 空輸入不會添加空任務
- [ ] 可以連續添加多個任務

### 刪除任務

- [ ] 點擊刪除按鈕，對應任務消失
- [ ] 刪除一個任務不影響其他任務
- [ ] 新添加的任務也能正常刪除

### 標記完成

- [ ] 點擊完成按鈕，任務有視覺變化（刪除線、變灰）
- [ ] 再次點擊可以取消完成狀態
- [ ] 完成/取消操作可以反覆進行
- [ ] 已完成的任務仍然可以刪除

### 整體檢查

- [ ] 按 F12 打開控制檯，沒有紅色報錯信息
- [ ] 三個功能可以配合使用，互不影響

## 發現問題了嗎？

現在試試：**刷新一下頁面**（按 F5 或 Ctrl+R）。

你會發現：所有任務都消失了。

這是正常的。目前我們的待辦清單隻"活"在瀏覽器的內存裏，就像在白板上寫字——關掉白板，字就沒了。

**下一輪，我們要解決這個問題**：讓數據能夠保存下來，即使刷新頁面、關閉瀏覽器，你的任務列表也不會丟失。

::: warning 💡 遇到問題？
如果某個功能不正常，先回到對應的小節查看"常見問題排查"。

如果是看不懂的報錯，可以先跳到 [4.5 Debug 實戰](../4.5-debug/) 學習如何讓 AI 幫你修復。

也可以繼續往下學，遇到問題隨時回來解決。
:::
