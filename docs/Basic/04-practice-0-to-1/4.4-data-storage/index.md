# 4.4 第三輪：讓數據活起來

> **本節目標**：讓任務數據保存下來，刷新頁面後還在

## 從 4.3 到 4.4：最後一塊拼圖

在 4.3 節，你已經讓待辦清單具備了完整的交互能力：添加、刪除、標記完成。

但如果你現在刷新一下頁面（按 F5），你會發現——**所有任務都消失了**。

這不是 Bug，而是因爲目前數據只存在於瀏覽器的"臨時記憶"中。就像在白板上寫字，擦掉就沒了。

本輪，我們要給待辦清單裝上"記憶"，讓它能記住你的任務。

## 本輪學習收穫

經過本輪學習，你將掌握：

- 理解爲什麼網頁需要數據存儲
- 使用 localStorage 實現數據持久化
- 驗證數據是否正確保存和讀取
- 完成一個「刷新不丟數據」的完整應用
