# 1.5.2 Step by Step 操作

接下來的每一步，我都會告訴你：
1. **做什麼** - 具體的操作
2. **看到什麼** - 你應該看到的結果
3. **檢查點** - 確認你做對了

跟着走就行。


## Step 1：打開工具（30 秒）

**做什麼：**

打開瀏覽器，訪問 [bolt.new](https://bolt.new)（國內用戶可用 [豆包 AI 編程](https://www.doubao.com/chat/coding)）

（或者你選擇的其他工具）

**你應該看到：**

一個簡潔的界面，中間有一個輸入框，讓你描述想要做什麼。

類似於 ChatGPT 的對話界面，但它專門用來生成網頁和應用。

**檢查點：**

- [ ] 界面已經加載完成
- [ ] 你能看到輸入框

::: warning 如果遇到問題
- **頁面加載很慢？** Bolt.new 在國內需要科學上網。建議國內用戶使用 [豆包 AI 編程](https://www.doubao.com/chat/coding)，訪問更流暢
- **需要登錄？** 大部分工具可以免費試用，用 Google 賬號或手機號登錄即可
:::


## Step 2：輸入 Prompt（30 秒）

現在，最關鍵的一步來了。

**做什麼：**

在輸入框裏，輸入下面這段話（可以直接複製）：

```
幫我做一個簡單的網頁，顯示一句勵志名言。

要求：
1. 頁面正中央顯示一句名言，比如"千里之行，始於足下"
2. 名言下方顯示作者名字
3. 頁面背景用淺色漸變，看起來舒服一點
4. 整體風格簡潔大方
```

然後按回車（或點擊發送按鈕）。

**你應該看到：**

AI 開始工作了！

它會：
- 思考一下你的需求
- 開始生成代碼
- 在右邊（或下方）即時顯示預覽效果

整個過程大概 10-30 秒。

**檢查點：**

- [ ] AI 正在生成內容
- [ ] 你能看到預覽畫面在變化

::: tip 小技巧
不用緊盯着代碼看。看不懂很正常——你只需要看最終的預覽效果就行。
:::


## Step 3：查看結果（1 分鐘）

**做什麼：**

等 AI 生成完畢後，看看預覽窗口裏的效果。

**你應該看到：**

一個漂亮的網頁，上面顯示着：
- 一句勵志名言（可能是"千里之行，始於足下"，也可能是 AI 自己選的其他名言）
- 作者名字
- 好看的背景色

**檢查點：**

- [ ] 頁面正常顯示，沒有報錯
- [ ] 能看到名言和作者
- [ ] 背景顏色是漸變的

::: info 恭喜！
如果你看到了上面這些，你已經成功做出了你的第一個 AI 網頁！

是的，就這麼簡單。
:::


## Step 4：嘗試修改（1 分鐘）

做出來不滿意？沒關係，**改它**。

**做什麼：**

在對話框裏繼續輸入你的修改意見。比如：

```
把名言換成"Stay hungry, stay foolish. —— Steve Jobs"
```

或者：

```
背景顏色改成深色系，字體顏色改成白色
```

或者：

```
加一個按鈕，點擊可以切換到下一句名言
```

**你應該看到：**

AI 會根據你的要求，修改代碼，預覽窗口也會即時更新。

**檢查點：**

- [ ] 修改生效了
- [ ] 頁面按你說的方式變化了


## 完整的 Prompt 參考

如果你想要一個更完整的版本，可以試試這個：

```
幫我做一個"每日一句"勵志名言網頁。

功能：
1. 頁面正中央顯示一句名言和作者
2. 有一個"換一句"按鈕，點擊可以隨機切換名言
3. 預設 5-10 句名言供切換

樣式：
1. 背景用柔和的漸變色（比如淺藍到淺紫）
2. 名言用大字體，居中顯示
3. 作者名用小一點的字體，顯示在名言下方
4. 按鈕要好看，有懸停效果
5. 整體風格現代、簡潔

請用 HTML + CSS + JavaScript 實現。
```

這個版本加入了交互功能（按鈕切換），你可以試試。


## 小結

你剛剛完成了什麼？

1. **打開工具** - 30 秒
2. **輸入需求** - 30 秒
3. **等待生成** - 30 秒
4. **查看結果** - 30 秒
5. **嘗試修改** - 1 分鐘

總共：**3 分鐘**。

你用自然語言"寫"了一個網頁。

沒有學任何編程語言。

沒有寫一行代碼。

**這就是 Vibe Coding。**
