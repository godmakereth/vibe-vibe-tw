# 1.5 Hello World：你的第一個 3 分鐘 AI 網頁

> **本節目標**：在 3 分鐘內，用 AI 做出你人生中第一個網頁。


還記得小時候第一次騎自行車的感覺嗎？

有人扶着後座，你踩下踏板，風吹過臉頰——那一刻你知道，你能行。

這一節，就是你的"第一次騎車"。

我們不講理論，不講原理。就是動手做一個東西出來。

**3 分鐘後，你會擁有一個真正能看、能用的網頁。**

不信？往下看。


## 爲什麼從 "Hello World" 開始？

在程序員的世界裏，有一個傳統：學任何新東西，第一個項目都叫 "Hello World"。

這不是什麼高深的東西，就是讓屏幕上顯示一句 "Hello World"（你好，世界）。

爲什麼這麼做？

因爲**做出第一個能跑的東西，比什麼都重要**。

它證明了一件事：你的環境是對的，你的方法是對的，你能做出東西。

剩下的，只是做得更復雜而已。
