## 爲什麼要分清這兩個概念？

很多人一上手就踩坑：

- ❌ 想做一個複雜的電商系統，直接 Vibe Coding → 結果改來改去，越改越亂
- ❌ 只是想做個簡單的小工具，卻花大量時間寫文檔 → 效率極低

**選對方法，事半功倍；選錯方法，事倍功半。**

別擔心，看完這一章，你就知道怎麼選了。
