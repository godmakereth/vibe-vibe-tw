## 學習路徑

本教程是整個體系的基礎版：

![學習路徑](/images/image-20251127115933477.png)

完成後可進入進階版，學習專業技術棧和完整產品開發流程。

## 從哪開始

```mermaid
flowchart TD
    A[開始] --> B{編程經驗？}
    B -->|從沒寫過代碼| C[零基礎入門]
    B -->|上過課/寫過一些| D{想怎麼學？}
    B -->|已是程序員| E{想怎麼學？}
    D -->|直接動手| F[跳到第4章實戰]
    D -->|系統學習| C
    E -->|先了解/系統學習| G[進階路徑]
    E -->|直接動手| F
    
    C --> H[第1章 → 第2章 → 第3章 → 第4章]
    F --> I[第4章 → 遇到問題回看]
    G --> J[1.2節 → 第3章 → 進階版]
```

| 背景 | 建議路徑 | 預計時長 |
|------|----------|----------|
| 零基礎 | 1→2→3→4 章順序讀 | 4-8 小時 |
| 文科/商科 | 重點 3、4 章 | 3-5 小時 |
| 理工科 | 1.2 節 → 3 章 → 進階版 | 2-4 小時 |
| 職場人士 | 2 章 → 4 章 | 3-4 小時 |

不確定？從第 1 章開始，覺得太基礎就往後跳。
