## 關於本教程

**Vibe Coding 基礎教程** 是爲零編程基礎的學習者設計的入門指南。無論你是文科生、設計師、產品經理，還是任何對創造軟件產品感興趣的人，這裏都是你的起點。

通過本教程，你將學會：

- **理解 Vibe Coding 的本質** —— 從"寫代碼"到"指揮 AI 寫代碼"
- **掌握 MVP 思維** —— 用最小成本驗證你的想法
- **學會與 AI 高效協作** —— 成爲 AI 的指揮官而非打字員
- **完成你的第一個項目** —— 從 0 到 1 實戰演練


## 章節概覽

| 章節 | 主題 | 核心內容 |
|------|------|----------|
| 0 | 序章 | 寫給不同的你、學習檢查清單、學習成果預告 |
| 1 | 覺醒 | 從碼農到指揮官、Vibe 與傳統開發對比、破除迷思、工具選擇、Hello World |
| 2 | 心法 | 不要加功能、MVP 思維、靈魂拷問、實戰案例 |
| 3 | 技術 | 提示詞工程、用戶旅程、第一份 PRD、優先級排序 |
| 4 | 從 0 到 1 實戰 | 環境搭建、靜態頁面、邏輯交互、數據存儲、調試指南 |
| 5 | 進階技能 | 版本控制、部署上線、安全意識、AI 侷限性 |
| 6 | 學習路徑 | 文科生路徑、理工生路徑、職場人路徑、資源推薦、持續學習 |


## 學習建議

::: tip 💡 給新手的建議
1. **按順序學習** —— 每個章節都建立在前一章的基礎上
2. **動手實踐** —— 光看不練假把式，一定要跟着做
3. **善用 AI** —— 遇到問題就問 AI，它是你最好的學習夥伴
4. **保持耐心** —— 學習曲線是真實存在的，但你一定能跨過去
:::
