# 附錄：常用 Prompt 模板庫

本附錄提供一系列可直接複製使用的 Prompt 模板，覆蓋項目創建、代碼修改、問題解決和文檔生成四大場景。

## 如何使用這份模板庫

1. **找到你的場景**：根據下方索引表，快速定位適合你的模板
2. **複製模板**：將模板複製到你的 AI 對話窗口
3. **替換佔位符**：把 `[方括號內容]` 替換成你的實際情況
4. **根據需要刪減**：如果某些字段不適用，可以刪除

## 模板使用前檢查清單

在發送 Prompt 之前，快速檢查：

- [ ] **背景信息完整嗎？** 項目類型、技術棧、當前進度
- [ ] **任務描述清晰嗎？** 具體要做什麼，輸入輸出是什麼
- [ ] **約束條件寫了嗎？** 必須遵守的限制，不能做什麼
- [ ] **佔位符都替換了嗎？** 沒有遺留 `[方括號]` 內容
- [ ] **期望格式說明了嗎？** 要完整代碼還是解釋
