# 3.7 本章總結與實戰演練

> **本節目標**：回顧本章核心內容，整合第二章與第三章的完整工作流，通過實戰演練內化技能。


## 經過本節學習，你將掌握

- 本章六大核心技能的精華總結與快速回顧
- 從「想清楚」到「說明白」的完整工作流
- 一份可隨時查閱的提示詞速查清單
- 四個綜合練習，幫助你內化所學技能
- 爲下一章實戰開發做好準備


## 爲什麼需要這一節

學完前面六節，你已經掌握了與 AI 對話的核心技能。但零散的知識點需要整合，才能形成真正可用的能力。

本節的任務是：

1. **回顧**：把六節內容壓縮成「記住什麼」
2. **整合**：把第二章的思考成果和第三章的表達技巧串聯起來
3. **實踐**：通過綜合練習，把知識轉化爲技能
4. **準備**：爲下一章的實戰開發做好準備


## 本節結構

| 小節 | 內容 | 你會得到 |
|-----|------|---------|
| 3.7.1 | 核心要點回顧 | 六大技能速查表、技巧選擇決策樹 |
| 3.7.2 | 完整工作流 | 從想法到代碼的八步流程、小李案例完整演示 |
| 3.7.3 | 提示詞速查清單 | 可打印的一頁紙速查表、分場景 Prompt 模板 |
| 3.7.4 | 實戰演練 | 四個綜合練習及參考答案 |
| 3.7.5 | 下一章預告 | 能力盤點、課後作業、第四章預告 |


## 學習建議

- **3.7.1-3.7.3**：快速瀏覽，收藏備用
- **3.7.4**：認真完成練習，對照參考答案自評
- **3.7.5**：完成課後作業，爲下一章做準備
