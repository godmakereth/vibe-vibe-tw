# 5.2.2 部署平臺選擇指南

部署平臺有很多，但對於初學者來說，選擇標準很簡單：**免費、簡單、能用**。

::: warning 中國大陸用戶請注意
Vercel 和 Netlify 的默認域名（`*.vercel.app` / `*.netlify.app`）在中國大陸被 DNS 污染，直接訪問會失敗或非常慢。

**建議**：大陸用戶優先使用 Zeabur，它的免費域名（`*.zeabur.app`）可以正常訪問。
:::

## 2025 年主流部署平臺對比

| 平臺 | 免費額度 | 大陸訪問 | 難度 | 最適合 |
|------|---------|---------|------|-------|
| **Zeabur** | $5/月額度 | 正常 | 簡單 | **大陸用戶首選** |
| **Vercel** | 100GB帶寬/月 | 需代理 | 簡單 | 海外用戶、Next.js 項目 |
| **Netlify** | 100GB帶寬/月 | 需代理 | 簡單 | 靜態網站 |
| **帽子云** | 完全免費 | 正常 | 簡單 | 大陸用戶、靜態網站 |
| **EdgeOne Pages** | 公測免費 | 正常 | 中等 | 騰訊雲用戶 |
| **GitHub Pages** | 完全免費 | 不穩定 | 中等 | 個人主頁、文檔站 |

## 2025 年免費額度詳情

### Zeabur（推薦）

- 免費試用計劃：每月 $5 額度
- 資源定價：內存約 $0.00025/GB-分鐘（約 $10.8/GB-月）
- vCPU：免費
- 網絡流量：$0.10/GB
- 特點：國人團隊、中文客服、支持支付寶

對於待辦清單這樣的小項目，免費額度完全夠用。

### Vercel

- Hobby 計劃（免費）：
  - 100GB 快速數據傳輸/月
  - 1M Edge 請求/月
  - 4 小時 CPU 時間/月
  - 最多 200 個項目
- 限制：僅限非商業用途

### Netlify

- 免費計劃：
  - 100GB 帶寬/月
  - 300 分鐘構建時間/月
  - 支持拖拽上傳

## 如何選擇

```
你在中國大陸嗎？
├── 是 → 使用 Zeabur（首選）或帽子云
└── 否 → 使用 Vercel（首選）或 Netlify
```

如果你不確定，跟着本教程使用 **Zeabur** 即可。它是目前對大陸用戶最友好的選擇，操作也足夠簡單。

## 本教程的選擇

| 場景 | 推薦平臺 | 原因 |
|------|---------|------|
| 大陸用戶 | Zeabur | 免費域名可正常訪問，中文界面 |
| 海外用戶 | Vercel | 最流行、功能強大、自動檢測 |
| 純靜態網站 | 帽子云 / Netlify | 完全免費、操作簡單 |
| 已有域名 | Vercel + Cloudflare | 性能最好 |

::: tip 不用糾結
對於你的第一個項目，選哪個平臺差別不大。先跑起來最重要，以後可以隨時換。
:::
