# 5.2 見世面：把網頁發到互聯網上

還記得你在第四章做的待辦清單嗎？它現在只能在你自己的電腦上運行。本節將帶你把它發佈到互聯網上，讓全世界都能訪問。

## 經過本節學習，你將掌握

- 理解"部署"的含義，消除對上線的恐懼
- 根據自己的情況選擇合適的部署平臺
- 完成從本地項目到線上網站的完整流程
- 獲得一個可以分享給任何人的真實網址

## 現在 vs 上線後

| 對比項 | 現在（本地） | 上線後 |
|-------|------------|-------|
| 訪問方式 | 只能在你電腦上打開 | 任何設備都能訪問 |
| 網址 | localhost:3000 | your-todo.zeabur.app |
| 分享 | 要把電腦遞給別人看 | 發個鏈接就行 |
| 手機使用 | 無法使用 | 隨時隨地使用 |
| 關機後 | 別人無法訪問 | 24小時在線 |
