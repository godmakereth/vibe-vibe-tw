# 5.2.6 大陸用戶的訪問解決方案

如果你在中國大陸，或者你的用戶在大陸，需要特別注意訪問問題。本節介紹幾種解決方案。

## 爲什麼會有訪問問題

Vercel、Netlify 等海外平臺的默認域名（如 `*.vercel.app`）被 DNS 污染，導致大陸用戶無法正常訪問或訪問很慢。

這不是你的代碼問題，而是網絡環境問題。

## 解決方案彙總

| 方案 | 難度 | 成本 | 推薦場景 |
|------|------|------|---------|
| 使用 Zeabur | 簡單 | 免費 | **首選方案** |
| 使用帽子云 | 簡單 | 免費 | 純靜態網站 |
| EdgeOne Pages | 中等 | 免費（公測） | 騰訊雲用戶 |
| 21YunBox | 簡單 | 免費 | Hugo 等靜態框架 |
| Cloudflare 代理 | 複雜 | 需購買域名 | 已用 Vercel 的進階用戶 |

## 方案一：直接用 Zeabur（推薦）

最簡單的方案。Zeabur 的免費域名 `*.zeabur.app` 在大陸可以正常訪問。

參見 [5.2.3 Zeabur 部署實戰](./5.2.3-zeabur.md)。

## 方案二：帽子云（純靜態網站）

帽子云是一個國產的靜態網站託管平臺，類似於國內版的 GitHub Pages。

**特點**：
- 完全免費
- 國內 CDN 節點，訪問速度快
- 支持 GitHub 自動部署
- 支持自定義域名

**適合**：純 HTML/CSS/JS 的靜態網站（比如我們的待辦清單）

**官網**：[maoziyun.com](https://www.maoziyun.com/)

### 使用步驟簡述

1. 註冊帽子云賬號
2. 連接你的 GitHub 倉庫
3. 選擇要部署的分支
4. 等待自動部署完成
5. 獲得 `*.maoziyun.com` 域名

## 方案三：EdgeOne Pages（騰訊雲）

EdgeOne Pages 是騰訊雲推出的前端部署平臺，目前處於公測階段，免費使用。

**特點**：
- 公測期免費
- 騰訊雲全球邊緣網絡加速
- 支持 Next.js、React、Vue 等現代框架
- 大陸訪問速度很快

**適合**：已有騰訊雲賬號的用戶，或需要更好國內訪問速度的項目

**官網**：[edgeone.ai](https://edgeone.ai/)

### 注意事項

EdgeOne Pages 的配置比 Zeabur 稍微複雜一些，需要熟悉騰訊雲的控制檯。如果你是第一次部署，建議先用 Zeabur。

## 方案四：21YunBox

21YunBox 是專門針對中國大陸用戶的託管平臺，對 Hugo、Jekyll 等靜態站點生成器支持很好。

**特點**：
- 國內 CDN 加速
- 支持 Gitee 部署（不需要 GitHub）
- 自動 SSL 證書
- 持續部署

**官網**：[21yunbox.com](https://www.21yunbox.com/)

## 方案五：Cloudflare 代理（進階）

如果你已經在 Vercel 上部署了項目，又想讓大陸用戶訪問，可以通過 Cloudflare 代理來解決。

### 前提條件

- 購買一個自己的域名（約 10-50 元/年）
- 註冊 Cloudflare 賬號（免費）

### 步驟概要

1. 在 Cloudflare 註冊賬號
2. 添加你的域名到 Cloudflare
3. 在域名註冊商處，將 DNS 服務器改爲 Cloudflare 提供的
4. 在 Vercel 中添加自定義域名
5. 在 Cloudflare 添加 CNAME 記錄指向 Vercel
6. 確保 Cloudflare 代理已開啓（橙色雲朵圖標）

### 這個方案的優缺點

| 優點 | 缺點 |
|------|------|
| 可以繼續使用 Vercel | 需要購買域名 |
| 有自己的專屬域名 | 配置步驟較多 |
| Cloudflare 有全球 CDN | 需要等待 DNS 生效 |

::: tip 基礎版建議
如果你剛開始學習，不建議現在折騰 Cloudflare 配置。先用 Zeabur 或帽子云把網站跑起來，等進階版再學習域名和 CDN 相關知識。
:::

## 如何選擇

```
你需要大陸訪問嗎？
├── 是
│   ├── 純靜態網站 → 帽子云（免費）或 Zeabur
│   ├── 需要服務端功能 → Zeabur
│   └── 已有騰訊雲賬號 → EdgeOne Pages
└── 否 → 直接用 Vercel / Netlify
```
