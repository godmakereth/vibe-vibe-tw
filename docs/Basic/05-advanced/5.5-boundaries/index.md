# 5.5 知邊界：Vibe Coding 的能與不能

> **本節目標**：清晰認識 Vibe Coding 的能力邊界，在擅長的領域充分發揮，在不擅長的領域知道何時尋求幫助。

經過本節學習，你將掌握：
- Vibe Coding 擅長解決哪類問題
- Vibe Coding 的真實侷限性在哪裏
- 如何判斷自己是否需要學習更多
- 與 AI 協作的正確心態

## 爲什麼要知道邊界

你可能會想："學這個幹嘛？知道能做什麼不就行了？"

這個想法很自然，但瞭解邊界其實是爲了**更好地發揮**，而不是限制你。

打個比方：你買了一輛代步車，它能讓你輕鬆在城市裏通勤。但如果你開着它去越野、去賽車，結果可能不太美好。知道它擅長什麼、不擅長什麼，才能把它用在最合適的地方。

Vibe Coding 也是如此：

| 知道邊界後 | 你可以 |
|-----------|--------|
| 瞭解擅長領域 | 在這些領域大膽嘗試，快速出成果 |
| 瞭解侷限性 | 遇到困難時不自我懷疑，知道是工具的問題 |
| 識別升級信號 | 在正確的時機決定是否深入學習 |
| 建立正確心態 | 把 AI 當工具而非魔法，保持掌控感 |
