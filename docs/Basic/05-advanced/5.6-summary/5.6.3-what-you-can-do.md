# 5.6.3 你現在能做什麼

經過本節閱讀，你將獲得：
- 完成基礎版後的能力清單
- 25+ 個按場景分類的項目建議
- 每個項目的核心挑戰和參考方法
- 30 天持續練習計劃

## 完成基礎版後，你具備的能力

| 能力維度 | 具體表現 | 可以做什麼 |
|---------|---------|-----------|
| **產品思維** | 能用 JTBD、MVP 思維規劃項目 | 清晰定義"爲誰做、做什麼、爲什麼" |
| **提示詞工程** | 能用 S.C.A.F.F. 框架與 AI 高效溝通 | 讓 AI 準確理解需求，減少返工 |
| **前端開發** | 能做出包含增刪改查的個人工具 | 靜態網站、表單交互、數據展示 |
| **本地存儲** | 能使用 localStorage 保存數據 | 數據刷新不丟失，換設備需重新輸入 |
| **版本管理** | 能保護代碼不丟失 | 大膽實驗，改崩了隨時回退 |
| **部署上線** | 能把作品發佈到互聯網 | 分享給朋友，獲得真實反饋 |
| **安全意識** | 知道如何避免常見的安全坑 | 保護自己的賬戶和隱私 |
| **迭代能力** | 能根據反饋持續改進 | 讓作品越來越好用 |

## 項目建議：按場景分類

以下項目都可以用基礎版學到的方法完成。挑一個你感興趣的開始吧。

### 場景一：個人效率工具

這類項目幫你管理時間、任務和個人事務。

| 項目 | 核心功能 | 學習價值 | 難度 |
|------|---------|---------|------|
| **番茄鍾** | 25分鐘倒計時 + 休息提醒 + 專注記錄 | 學習定時器和時間處理 | ⭐⭐ |
| **習慣打卡器** | 每日打卡 + 連續天數 + 日曆視圖 | 學習日期處理和數據統計 | ⭐⭐ |
| **簡易記賬本** | 收支記錄 + 分類統計 + 月度彙總 | 學習數字計算和數據展示 | ⭐⭐ |
| **讀書筆記** | 書籍管理 + 筆記記錄 + 標籤分類 | 學習文本處理和搜索過濾 | ⭐⭐⭐ |
| **密碼生成器** | 自定義長度 + 字符類型選擇 + 一鍵複製 | 學習隨機數和剪貼板操作 | ⭐ |
| **倒計時工具** | 目標日期設置 + 剩餘天數顯示 | 學習日期差值計算 | ⭐ |
| **每日一句** | 隨機名言 + 收藏功能 + 分享按鈕 | 學習數組隨機和分享API | ⭐ |

**參考方法**：使用待辦清單的開發流程，先用 S.C.A.F.F. 框架描述需求，再分層實現。

### 場景二：數據分析可視化

這類項目幫你整理和展示數據。雖然基礎版不涉及複雜的數據庫，但可以用本地數據做簡單的分析展示。

| 項目 | 核心功能 | 學習價值 | 難度 |
|------|---------|---------|------|
| **個人支出分析** | 手動輸入數據 + 餅圖/柱狀圖展示 | 學習圖表庫使用（如 Chart.js） | ⭐⭐⭐ |
| **閱讀統計** | 閱讀時長記錄 + 周/月彙總圖表 | 學習時間統計和數據聚合 | ⭐⭐ |
| **體重追蹤** | 每日記錄 + 趨勢折線圖 | 學習折線圖綁定數據 | ⭐⭐ |
| **學習時間統計** | 科目分類 + 時長記錄 + 佔比分析 | 學習多維度數據處理 | ⭐⭐⭐ |
| **問卷結果展示** | 靜態數據輸入 + 可視化報告頁 | 學習數據可視化基礎 | ⭐⭐ |

**參考方法**：讓 AI 幫你集成 Chart.js 等圖表庫，你只需要提供數據結構和展示需求。

### 場景三：自動化腳本/批處理

這類項目幫你減少重複勞動。基礎版主要做瀏覽器端的自動化，複雜的本地文件操作需要進階版。

| 項目 | 核心功能 | 學習價值 | 難度 |
|------|---------|---------|------|
| **文本格式轉換** | 文本輸入 + 格式轉換（大小寫、去空格等） | 學習字符串處理 | ⭐ |
| **JSON 格式化** | JSON 輸入 + 美化/壓縮 + 語法檢查 | 學習 JSON 解析和格式化 | ⭐⭐ |
| **Markdown 預覽器** | 即時編輯 + 即時預覽 | 學習 Markdown 解析庫 | ⭐⭐ |
| **鏈接批量提取** | 粘貼文本 + 自動提取所有鏈接 | 學習正則表達式基礎 | ⭐⭐ |
| **單位換算器** | 長度/重量/溫度等單位轉換 | 學習數學計算和單位換算 | ⭐ |

**參考方法**：這類工具通常輸入輸出明確，非常適合用 Vibe Coding 快速實現。

### 場景四：給家人/朋友做的工具

這類項目幫你解決身邊人的實際問題，是獲得真實反饋的最佳方式。

| 項目 | 核心功能 | 適合誰 | 難度 |
|------|---------|-------|------|
| **家庭菜單規劃** | 每週菜單 + 食材清單生成 | 負責做飯的家人 | ⭐⭐ |
| **藥物提醒** | 用藥記錄 + 定時提醒 | 需要定期服藥的長輩 | ⭐⭐ |
| **生日提醒** | 親友生日管理 + 倒計時提醒 | 容易忘記重要日子的人 | ⭐⭐ |
| **簡易相冊** | 圖片上傳展示 + 時間軸排列 | 想分享照片的家人 | ⭐⭐⭐ |
| **購物清單** | 共享清單 + 已購標記 | 一起生活的家人 | ⭐⭐ |
| **家庭記賬本** | 共同支出記錄 + 分類統計 | 合租室友或小家庭 | ⭐⭐⭐ |

**參考方法**：先和使用者聊聊他們的痛點（用靈魂三問），再用 MVP 思維確定核心功能。

### 場景五：個人展示/創意項目

這類項目展示你的個人品牌或創意想法。

| 項目 | 核心功能 | 學習價值 | 難度 |
|------|---------|---------|------|
| **個人主頁** | 自我介紹 + 作品集展示 + 聯繫方式 | 學習頁面佈局和樣式設計 | ⭐⭐ |
| **在線簡歷** | 響應式簡歷頁面 + 下載PDF功能 | 學習響應式設計和打印樣式 | ⭐⭐ |
| **作品集網站** | 項目展示 + 詳情頁 + 技術棧說明 | 整合所有學習成果 | ⭐⭐⭐ |
| **小遊戲：猜數字** | 隨機數生成 + 猜測反饋 + 記錄次數 | 學習遊戲邏輯設計 | ⭐ |
| **小遊戲：記憶翻牌** | 卡片匹配 + 計時 + 最佳記錄 | 學習狀態管理和動畫 | ⭐⭐⭐ |
| **表白網頁** | 動畫效果 + 音樂播放 + 個性化內容 | 學習CSS動畫和音頻API | ⭐⭐ |

**參考方法**：創意項目不需要複雜功能，重點是視覺效果和交互體驗。讓 AI 幫你實現炫酷的 CSS 動畫。

## 如何選擇你的下一個項目

不知道選哪個？用這個決策流程：

```
你現在最想解決什麼問題？
│
├── 工作/學習效率低 → 番茄鍾、習慣打卡器
├── 數據雜亂無章 → 記賬本、閱讀統計
├── 重複勞動太多 → 文本格式轉換、鏈接提取
├── 想幫助家人朋友 → 藥物提醒、生日提醒
└── 想展示自己 → 個人主頁、在線簡歷
```

**核心原則**：選一個你真正會用的項目，而不是"看起來最酷"的項目。

## 30 天練習計劃

如果你想持續提升，可以嘗試這個計劃：

| 周次 | 目標 | 建議項目 |
|------|------|---------|
| 第 1 周 | 複習鞏固 | 給待辦清單添加 2-3 個新功能 |
| 第 2 周 | 獨立完成 | 選一個 ⭐ 難度的項目從零做起 |
| 第 3 周 | 挑戰進階 | 選一個 ⭐⭐ 難度的項目 |
| 第 4 周 | 分享展示 | 整理作品集，寫 README，分享給朋友 |

**每個項目的標準流程**：
1. 用靈魂三問定義需求
2. 用 S.C.A.F.F. 框架寫第一個 Prompt
3. 分層迭代實現功能
4. 部署上線，收集反饋
5. 整理成作品集

- 選項目的關鍵是"你真正會用"，而不是"看起來最酷"
