# 附錄：心法篇補充材料

本附錄爲第二章「心法篇」提供補充材料，幫助你更好地理解和應用七個思維模型。


## 附錄內容導航

| 附錄 | 內容 | 適合誰 |
|-----|------|-------|
| [A. 思維模型速查表](./A-templates.md) | 七個模型的模板彙總，帶填寫示例 | 需要快速查閱模板的讀者 |
| [B. 擴展案例庫](./B-case-studies.md) | 6 個真實案例的思維模型應用 | 想看更多實例的讀者 |
| [C. 常見誤區診斷](./C-mistakes.md) | 七個模型的典型誤區與改進建議 | 想自檢和避坑的讀者 |
| [D. 深度參考資源](./D-references.md) | 框架出處、推薦書單、在線資源 | 想深入學習的讀者 |


## 如何使用本附錄

**場景一：正在做項目，需要快速查模板**
→ 直接翻閱 [A. 思維模型速查表](./A-templates.md)

**場景二：想看看別人是怎麼用這些思維模型的**
→ 閱讀 [B. 擴展案例庫](./B-case-studies.md)

**場景三：感覺自己哪裏沒想清楚，但不知道問題在哪**
→ 用 [C. 常見誤區診斷](./C-mistakes.md) 自檢

**場景四：想深入瞭解這些框架的理論背景**
→ 參考 [D. 深度參考資源](./D-references.md)


## 與正文的關係

本附錄是正文的**補充**，而非替代。

- 如果你還沒讀過正文，建議先閱讀第二章各節內容
- 附錄提供的是「工具性」材料：模板、案例、誤區清單、參考資源
- 正文提供的是「理解性」內容：爲什麼需要這些思維模型、如何思考

**建議的使用方式**：先讀正文理解原理，再用附錄輔助實踐。
