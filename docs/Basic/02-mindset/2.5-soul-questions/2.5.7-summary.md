# 2.5.7 本節核心要點

## 靈魂三問速記

在動手之前，花3分鐘回答：

```
✓ 用戶是誰——具體到能給TA發微信、能叫出TA的名字
✓ 痛點在哪——追問到能感受到情緒、能說出發生頻率
✓ 爲什麼選你——找到與現有方案的差異，或證明值得自己做
```


## 紅綠燈法則

- 🟢 三個問題都能一句話回答 → 可以開始
- 🟡 有的問題說不清楚 → 需要再想想
- 🔴 有的問題完全答不上來 → 先停下來

**回答不清楚就動手，做出來的東西大概率沒人用（包括自己）。**


## 不同場景的核心問法

| 場景 | 用戶是誰 | 痛點在哪 | 爲什麼選你 |
|-----|---------|---------|-----------|
| 做產品 | 用這個產品的人 | TA完成任務時的障礙 | 你的差異化價值 |
| 數據分析 | 看報告的決策者 | TA決策時缺什麼信息 | 你選擇這種呈現方式的理由 |
| 自動化 | 自己 | 什麼重複勞動在消耗你 | 投入產出是否划算 |
| 給家人做 | 那個家人 | 現有工具哪裏不適合TA | 你能專門爲TA適配什麼 |


## 下一步

完成靈魂三問自檢後，你已經準備好進入下一節：把這些思考應用到不同場景中。

如果你的三個問題都是綠燈，恭喜你——你比80%的人都更清楚自己在做什麼。
