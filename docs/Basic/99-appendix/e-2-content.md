# E.2 效率工具類（10 個）

這些項目能真正提高你的工作效率。做出來之後，你會每天都用。


## 1. Markdown 預覽器

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：Markdown 解析、即時預覽

**功能描述**：
- 左側輸入 Markdown
- 右側即時顯示渲染效果
- 支持常用 Markdown 語法
- 可選：導出爲 HTML

**學到什麼**：第三方庫使用（如 marked.js）、分欄佈局


## 2. JSON 格式化工具

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：JSON 解析、代碼高亮

**功能描述**：
- 粘貼混亂的 JSON
- 一鍵格式化/美化
- 語法高亮顯示
- 錯誤提示

**學到什麼**：JSON.parse/stringify、錯誤處理


## 3. 單位轉換器

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：數學計算、單位映射

**功能描述**：
- 支持長度、重量、溫度等單位轉換
- 輸入一個值，顯示其他單位的結果
- 界面簡潔易用

**學到什麼**：數值處理、select 下拉框


## 4. 顏色工具箱

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：顏色格式轉換、顏色選擇器

**功能描述**：
- HEX/RGB/HSL 互轉
- 顏色選擇器
- 顯示顏色預覽
- 複製顏色代碼

**學到什麼**：顏色格式、input type="color"


## 5. 文本統計工具

**難度**：⭐（入門）  
**預計時長**：30 分鐘 - 1 小時  
**核心技術點**：字符串處理

**功能描述**：
- 統計字數、字符數
- 統計行數、段落數
- 統計單詞數（英文）
- 即時更新

**學到什麼**：字符串方法、正則表達式基礎


## 6. 二維碼生成器

**難度**：⭐⭐（基礎）  
**預計時長**：1 小時  
**核心技術點**：第三方庫使用

**功能描述**：
- 輸入文字或鏈接
- 生成二維碼圖片
- 可下載保存
- 可選：自定義顏色

**學到什麼**：使用 qrcode.js 等第三方庫


## 7. 剪貼板歷史

**難度**：⭐⭐⭐（進階）  
**預計時長**：2-3 小時  
**核心技術點**：剪貼板 API、歷史記錄

**功能描述**：
- 記錄複製過的內容
- 點擊快速複製回去
- 搜索歷史記錄
- 可清除歷史

**學到什麼**：Clipboard API、列表管理


## 8. 快速筆記（便籤）

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：拖拽、LocalStorage

**功能描述**：
- 創建多個便籤
- 可拖動位置
- 不同顏色區分
- 自動保存

**學到什麼**：拖拽 API、位置存儲


## 9. 快捷短語管理

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：文本管理、快捷複製

**功能描述**：
- 保存常用短語/模板
- 分類整理
- 一鍵複製
- 搜索

**學到什麼**：分類管理、剪貼板操作


## 10. 網頁截圖標註

**難度**：⭐⭐⭐（進階）  
**預計時長**：3-4 小時  
**核心技術點**：Canvas、圖片處理

**功能描述**：
- 上傳截圖
- 添加箭頭、框、文字標註
- 導出標註後的圖片

**學到什麼**：Canvas 綁定、圖片編輯基礎


## 效率工具的共同特點

好的效率工具應該：
- **快**：打開就能用，不需要登錄
- **簡**：功能聚焦，不堆砌
- **存**：數據本地保存，不丟失

::: tip 做效率工具的心法
先問自己：「我每天重複做的事情有哪些？」然後把它做成工具。**解決自己的問題，就是最好的項目 idea。**
:::
