# D.2 IDE 集成式對比

IDE 集成式工具把 AI 能力直接嵌入代碼編輯器，可以在寫代碼的同時獲得 AI 幫助。更適合有一定基礎、想深入學習的用戶。


## 綜合對比表

| 工具 | 基於 | 價格 | 特點 | 適合人羣 |
|------|------|------|------|----------|
| **Cursor** | VS Code | $20/月 | 最強 AI IDE | 想深度使用 AI 的開發者 |
| **GitHub Copilot** | VS Code 插件 | $10/月 | 代碼補全王者 | 已有 VS Code 經驗 |
| **Windsurf** | VS Code | 免費版可用 | 輕量好用 | 想免費體驗的用戶 |
| **Cline** | VS Code 插件 | 按 API 付費 | 開源靈活 | 喜歡自定義的用戶 |
| **Trae** | 獨立 IDE | 免費 | 字節出品 | 國內用戶 |
| **Kiro** | VS Code | 免費 | 亞馬遜出品，Spec 驅動 | 想體驗 Spec Coding |


## 各工具詳解

### Cursor

**官網**：cursor.com

**是什麼**：基於 VS Code 的 AI 原生 IDE，被稱爲「AI 編程的未來」。

**核心功能**：
- **Tab 補全**：按 Tab 接受 AI 的代碼建議
- **Cmd+K**：選中代碼後，告訴 AI 怎麼改
- **Chat**：在側邊欄和 AI 對話
- **Composer**：讓 AI 同時編輯多個文件

**價格**：
- 免費版：有限次數
- Pro：$20/月

**優點**：
- AI 集成最深
- 功能最全面
- 社區活躍

**缺點**：
- 付費才能充分體驗
- 新手可能有點複雜


### GitHub Copilot

**官網**：github.com/features/copilot

**是什麼**：GitHub 推出的 AI 編程助手，作爲 VS Code 插件使用。

**核心功能**：
- **代碼補全**：寫代碼時自動給建議
- **Chat**：在 VS Code 裏和 AI 對話
- **解釋代碼**：選中代碼讓 AI 解釋

**價格**：
- 個人：$10/月
- 學生：免費

**優點**：
- 代碼補全體驗絲滑
- 和 GitHub 生態無縫整合
- 學生可以免費用

**缺點**：
- 只是插件，功能相對簡單
- 需要科學上網


### Windsurf

**官網**：codeium.com/windsurf

**是什麼**：Codeium 推出的 AI IDE，主打免費好用。

**核心功能**：
- 類似 Cursor 的 AI 對話
- 代碼補全
- Agent 模式（自動執行多步操作）

**價格**：
- 基礎功能免費
- Pro 版付費

**優點**：
- 免費版就很能用
- 輕量流暢
- 學習成本低

**缺點**：
- 功能不如 Cursor 全面
- 社區相對小


### Cline

**官網**：github.com/cline/cline

**是什麼**：開源的 VS Code AI 插件，可以用自己的 API Key。

**核心功能**：
- 在 VS Code 中和 AI 對話
- 可以選擇不同的 AI 模型
- Agent 模式

**價格**：
- 軟件免費
- 按 API 調用付費

**優點**：
- 開源透明
- 可以用便宜的模型
- 高度可定製

**缺點**：
- 需要自己配置 API Key
- 不適合完全新手


### Trae

**官網**：trae.ai

**是什麼**：字節跳動推出的 AI IDE，國內可直接使用。

**核心功能**：
- AI 對話和代碼生成
- 代碼補全
- 項目理解

**價格**：免費

**優點**：
- 國內可用
- 完全免費
- 中文支持好

**缺點**：
- 功能在快速迭代中
- 生態還在建設


### Kiro

**官網**：kiro.dev

**是什麼**：亞馬遜推出的 Spec 驅動 AI IDE。

**核心功能**：
- **Spec 模式**：先寫需求文檔，再生成代碼
- 自動生成 requirements.md、design.md、tasks.md
- 按任務清單逐步開發

**價格**：免費預覽中

**優點**：
- 體驗 Spec Coding 的好選擇
- 適合複雜項目
- 文檔和代碼結合

**缺點**：
- 還在早期階段
- 學習曲線較陡


## 選擇建議

| 你的情況 | 推薦工具 |
|----------|----------|
| 零基礎，想先試試 | Windsurf（免費） |
| 有 VS Code 經驗 | GitHub Copilot |
| 想深度使用 AI | Cursor |
| 國內用戶 | Trae |
| 想體驗 Spec Coding | Kiro |
| 喜歡折騰 | Cline |

::: warning 提醒
這些工具需要一定的編程基礎才能充分發揮作用。如果你是完全零基礎，建議先用對話式 AI 或專用 AI IDE（下一節）。
:::
