# E.4 數據可視化類（10 個）

把數據變成圖表，讓信息一目瞭然。這類項目非常適合做數據分析、做報告的人。


## 1. 個人支出餅圖

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：Chart.js 使用、數據聚合

**功能描述**：
- 輸入支出數據
- 按分類生成餅圖
- 顯示各分類佔比
- 可選擇時間範圍

**學到什麼**：圖表庫使用、數據處理


## 2. 體重/健康追蹤圖

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：折線圖、日期數據

**功能描述**：
- 每天記錄體重/步數等
- 折線圖展示趨勢
- 顯示平均值、最高/最低
- 目標線標註

**學到什麼**：時間序列數據可視化


## 3. 習慣熱力圖

**難度**：⭐⭐⭐（進階）  
**預計時長**：2-3 小時  
**核心技術點**：熱力圖、日曆視圖

**功能描述**：
- 類似 GitHub 貢獻圖
- 顏色深淺表示完成程度
- 點擊查看當天詳情
- 統計連續天數

**學到什麼**：熱力圖實現、CSS Grid


## 4. 時間分配統計

**難度**：⭐⭐⭐（進階）  
**預計時長**：2-3 小時  
**核心技術點**：計時器、環形圖

**功能描述**：
- 記錄時間花在哪裏
- 分類統計（工作/學習/娛樂等）
- 環形圖展示佔比
- 周/月統計

**學到什麼**：時間追蹤、多種圖表


## 5. 投票/調查結果展示

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：柱狀圖、即時更新

**功能描述**：
- 創建投票選項
- 用戶投票
- 即時顯示投票結果
- 柱狀圖可視化

**學到什麼**：即時數據更新、柱狀圖


## 6. 項目進度儀表盤

**難度**：⭐⭐⭐（進階）  
**預計時長**：2-3 小時  
**核心技術點**：進度條、多圖表組合

**功能描述**：
- 項目任務列表
- 完成進度百分比
- 多維度統計
- 儀表盤佈局

**學到什麼**：儀表盤設計、多圖表佈局


## 7. 簡易股票/幣價圖表

**難度**：⭐⭐⭐（進階）  
**預計時長**：2-3 小時  
**核心技術點**：API 調用、即時數據

**功能描述**：
- 輸入股票/加密貨幣代碼
- 顯示價格走勢圖
- K 線圖基礎版
- 自動刷新

**學到什麼**：API 調用、金融圖表


## 8. 學習時長統計

**難度**：⭐⭐（基礎）  
**預計時長**：1-2 小時  
**核心技術點**：計時器、統計圖表

**功能描述**：
- 學習計時（開始/結束）
- 分科目記錄
- 日/周/月統計
- 圖表展示

**學到什麼**：時間統計、多維度展示


## 9. 目標完成雷達圖

**難度**：⭐⭐⭐（進階）  
**預計時長**：2-3 小時  
**核心技術點**：雷達圖、多維度數據

**功能描述**：
- 設定多個目標維度
- 給每個維度打分
- 雷達圖展示完成情況
- 對比不同時期

**學到什麼**：雷達圖實現、多維度分析


## 10. 簡易數據看板

**難度**：⭐⭐⭐（進階）  
**預計時長**：3-4 小時  
**核心技術點**：多圖表、佈局設計

**功能描述**：
- 數字卡片（總數、增長率等）
- 多個圖表組合
- 響應式佈局
- 可選擇時間範圍

**學到什麼**：儀表盤設計、數據大屏基礎


## 數據可視化常用圖表庫

| 庫名 | 特點 | 適合場景 |
|------|------|----------|
| Chart.js | 簡單易用 | 入門首選 |
| ECharts | 功能強大 | 複雜圖表 |
| D3.js | 靈活度最高 | 高度定製 |
| ApexCharts | 現代美觀 | 追求顏值 |

::: tip 建議
先用 **Chart.js** 入門，它的文檔清晰，上手最快。等熟悉了再嘗試其他庫。
:::


## 數據可視化的核心原則

1. **清晰**：一眼能看懂在說什麼
2. **準確**：數據不能誤導
3. **美觀**：顏色協調、佈局合理
4. **交互**：鼠標懸停顯示詳情

::: warning 注意
做數據可視化項目時，先用假數據把圖表做出來，最後再接真實數據。這樣迭代更快。
:::
