# 8.4.3 電腦垃圾和編輯器配置——系統與 IDE 文件

每個人的電腦和編輯器都不一樣——這些"個人偏好"不該污染代碼倉庫。

## 操作系統文件

### macOS

```gitignore
# macOS 文件夾元數據
.DS_Store
.AppleDouble
.LSOverride

# 圖標文件
Icon?

# 廢紙簍
.Trashes

# Spotlight 索引
.Spotlight-V100
```

### Windows

```gitignore
# 縮略圖緩存
Thumbs.db
ehthumbs.db
ehthumbs_vista.db

# 桌面配置
Desktop.ini

# 快捷方式
*.lnk

# 回收站
$RECYCLE.BIN/
```

### Linux

```gitignore
# 備份文件
*~

# 臨時文件
.fuse_hidden*
.nfs*
```

## IDE 配置

### VS Code

```gitignore
# 用戶設置（個人偏好）
.vscode/settings.json

# 工作區配置
.vscode/*.code-workspace

# 歷史記錄
.history/

# 但可能要保留這些共享配置：
# !.vscode/extensions.json      # 推薦擴展
# !.vscode/launch.json          # 調試配置
# !.vscode/tasks.json           # 任務配置
```

### JetBrains (WebStorm/IntelliJ)

```gitignore
# IntelliJ IDEA 項目配置
.idea/

# 但可能保留代碼風格：
# !.idea/codeStyles/
# !.idea/inspectionProfiles/
```

### Vim/Neovim

```gitignore
# 交換文件
*.swp
*.swo
*~

# Session 文件
Session.vim
Sessionx.vim
```

### Emacs

```gitignore
# 備份
*~
\#*\#
.#*

# 自動保存
.saves-*
```

## 全局 gitignore

與其在每個項目都配置系統文件，不如設置全局忽略：

### 配置方法

```bash
# 1. 創建全局 gitignore 文件
touch ~/.gitignore_global

# 2. 告訴 Git 使用這個文件
git config --global core.excludesFile ~/.gitignore_global
```

### 推薦的全局配置

```gitignore
# ~/.gitignore_global

# macOS
.DS_Store
.AppleDouble

# Windows
Thumbs.db
Desktop.ini

# 編輯器
*.swp
*.swo
*~
.idea/
.vscode/settings.json

# 個人筆記
TODO.md
NOTES.md
```

## 項目共享配置

有些 IDE 配置可能需要團隊共享：

### 推薦擴展 (.vscode/extensions.json)

```json
{
  "recommendations": [
    "dbaeumer.vscode-eslint",
    "esbenp.prettier-vscode",
    "prisma.prisma"
  ]
}
```

### 調試配置 (.vscode/launch.json)

```json
{
  "version": "0.2.0",
  "configurations": [
    {
      "type": "node",
      "request": "launch",
      "name": "Next.js: debug",
      "runtimeExecutable": "pnpm",
      "runtimeArgs": ["dev"],
      "port": 9229
    }
  ]
}
```

## 配置策略

| 文件 | 建議 | 原因 |
|------|------|------|
| .vscode/settings.json | 忽略 | 個人偏好 |
| .vscode/extensions.json | 提交 | 統一開發環境 |
| .vscode/launch.json | 提交 | 共享調試配置 |
| .idea/ | 忽略 | IDE 特定 |
| .editorconfig | 提交 | 跨 IDE 格式化 |

## EditorConfig 替代方案

使用 `.editorconfig` 可以跨 IDE 統一基本格式：

```ini
# .editorconfig
root = true

[*]
charset = utf-8
end_of_line = lf
indent_style = space
indent_size = 2
insert_final_newline = true
trim_trailing_whitespace = true

[*.md]
trim_trailing_whitespace = false
```
