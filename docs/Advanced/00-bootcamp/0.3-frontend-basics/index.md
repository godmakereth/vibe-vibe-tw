# 0.3 前端基礎：構建用戶眼中的世界

歡迎來到前端開發的核心地帶。

如果說後端是餐廳的廚房，負責處理食材、烹飪佳餚，那麼前端就是餐廳的大堂——它是顧客（用戶）直接體驗、感受和交互的地方。前端開發的目標，就是創造出美觀、易用、響應迅速的用戶界面。

對於 Vibe Coder 而言，前端不僅僅是“畫頁面”，更是將產品邏輯、數據和用戶體驗融爲一體的藝術。在 AI 的加持下，我們可以更快地將想法轉化爲可交互的原型，並將其打磨成精緻的產品。

在本章中，你將掌握前端開發的三駕馬車：

*   **HTML (超文本標記語言)**：網頁的“骨架”，負責定義頁面的結構和內容。
*   **CSS (層疊樣式表)**：網頁的“外貌”，負責設計頁面的樣式、佈局和美感。
*   **JavaScript (JS)**：網頁的“靈魂”，負責實現頁面的交互邏輯和動態功能。

同時，我們還會探討一個至關重要的工程思想：

*   **關注點分離**：如何讓結構、樣式和行爲各司其職，使代碼更清晰、更易於維護。

準備好，讓我們一起開始構建用戶能看到、能觸摸、能感受到的數字世界。
