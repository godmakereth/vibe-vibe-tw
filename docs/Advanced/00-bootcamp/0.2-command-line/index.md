# 0.2 命令行：與計算機的直接對話

歡迎來到命令行的世界！

如果說圖形用戶界面（GUI）是與計算機進行“菜單式”的互動，那麼命令行界面（CLI）就是與計算機進行“對話式”的交流。你不再需要通過點擊圖標和按鈕來操作，而是通過輸入精確的文本指令，直接告訴計算機你想要做什麼。

對於 Vibe Coder 來說，命令行不是一個過時的工具，而是一個高效、強大且可自動化的核心技能。在 AI 輔助編程的時代，我們更需要通過命令行來精準地控制開發環境、執行腳本、管理服務器。

在本章中，你將學到：

*   **文件系統導航**：如何在目錄樹中自由穿梭。
*   **文件基本操作**：如何創建、查看、移動和刪除文件與目錄。
*   **權限管理**：如何用命令行控制誰能動你的文件。
*   **跨平臺差異**：理解 Windows (PowerShell) 和 macOS/Linux (Bash/Zsh) 之間的常用命令區別。

準備好，讓我們開始學習這門計算機的“通用語言”。
