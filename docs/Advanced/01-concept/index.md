# 1 ｜概念與準備（從思維到工具）

本章是從"瞭解概念"到"動手實踐"的過渡階段。在 Bootcamp 中，你已經建立了對全棧開發和 Vibe Coding 的基本認知；現在，我們要把這些認知轉化爲**可執行的行動**。

### 本章目標

完成本章學習後，你將能夠：

1. **搭建完整的開發環境**：擁有一套可以立即開始編碼的工具鏈（IDE、Node.js、Git）
2. **跑通第一個 Next.js 項目**：親眼看到 Hello World 頁面在瀏覽器中運行
3. **建立 Vibe Coding 心法**：理解如何從"寫代碼"轉變爲"指揮 AI 寫代碼"
4. **完成首次部署**：讓你的作品擁有一個全世界都能訪問的 URL
5. **掌握 AI 協作工作流**：學會如何與 AI 高效配合，而非被 AI 牽着走

### 章節導航

| 小節 | 主題 | 你將學到 |
|------|------|----------|
| 1.1 | 工具裝配與驗證 | 選擇 IDE、配置 AI 助手、初始化 Next.js 項目 |
| 1.2 | Vibe Coding 心法 | 思維轉變、提示工程、代碼審查、模型選擇 |
| 1.3 | 最小上線實戰 | 用 1Panel 完成第一次部署 |
| 1.4 | AI 協作工作流 | 建立可複用的人機協作模式 |
| 1.5 | 工具鏈全景 | IDE/Git/數據庫/部署平臺的全面配置 |

### 學習建議

本章內容偏實操，建議你：

- **邊看邊做**：每個小節都有實際操作步驟，動手比看更重要
- **不要追求完美**：先跑通最小閉環，再迭代優化
- **遇到問題找 AI**：這正是 Vibe Coding 的核心——用自然語言描述問題，讓 AI 幫你解決
