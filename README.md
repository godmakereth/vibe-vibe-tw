### 核心理念

踐行 OpenAI 聯合創始人 **Andrej Karpathy** 提出的 Vibe Coding 理念——**從 Coder 到 Commander**：通過自然語言與 AI 對話，讓編程從"寫代碼"轉變爲"對話式創作"。

> *"完全沉浸於編程的'氛圍'中，忘記代碼的存在。"*

### 快速開始

📖 **在線閱讀**：[www.vibevibe.cn](https://www.vibevibe.cn)

| 你是誰 | 推薦起點 |
|-------|---------|
| 完全零基礎 | 基礎篇 → 第1章 覺醒 |
| 用過 ChatGPT 等大語言模型工具但沒做過項目 | 基礎篇 → 第2章 心法 |
| 有編程基礎想學 Vibe Coding | 基礎篇快速瀏覽 → 進階篇 |
| 想直接動手做項目 | 基礎篇 → 第4章 實戰 |
| 想找項目練手 | 實踐篇 |

### 教程定位

國內首個系統化的 Vibe Coding 開源教程，分爲四大板塊：

| 板塊 | 定位 | 適合人羣 |
|-----|------|---------|
| 📘 **基礎篇** | AI 編程入門 + 心法 + 第一個項目 | 完全零基礎、用過 ChatGPT 但沒做過項目 |
| 📗 **進階篇** | 全棧技術棧 + 工程化 + 生產級部署 | 想深入學習、有一定編程基礎想轉型 |
| 📙 **實踐篇** | 分人羣項目實戰 + 進階技能訓練 | 想通過動手練習鞏固所學 |
| 📕 **優質文章篇** | 精選學習資源 + 行業前沿追蹤 | 想持續學習、保持行業敏感度 |

- **爲什麼需要這份教程？** Vibe Coding 已成爲全球趨勢（越來越多的創業公司大部分代碼由 AI 輔助生成），但國內學習資源碎片化嚴重，缺乏系統化、零基礎友好、包含完整進階路徑的教程。

- **適合人羣**：編程零基礎者、大學生、創業者、傳統程序員、職場人士——任何想用 AI 快速實現想法、提升工作效率或學習新範式的人。

### 學習產出

| 階段 | 你將獲得 |
|-----|---------|
| **基礎篇** | 掌握 Vibe Coding 心法 · 用自然語言指揮 AI · 獨立完成項目 · 部署上線能力 |
| **進階篇** | 現代全棧技術棧 · 前後端架構 · 用戶系統開發 · 生產級部署 · 工程化思維 |
